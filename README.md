## ✨ Fitur

| Fitur | Deskripsi |
|-------|-----------|
| 🌸 **Romantis** | Background bunga & hati, tema rose pink |
| 🌙 **Dark Mode** | Toggle gelap/terang di pengaturan |
| 💞 **Pairing** | Hubungkan akun dengan pasangan via kode |
| ⏰ **Reminder** | Ingatkan pasangan untuk makan, minum obat, dll |
| 📸 **Bukti Foto** | Kirim foto sebagai bukti sudah melakukan |
| 📋 **Riwayat** | Lihat riwayat reminder hari ini |

## 🛠 Tech Stack

- **Framework:** Expo (React Native) SDK 54
- **Language:** TypeScript
- **Backend:** Supabase
- **Auth:** Supabase Auth
- **Storage:** Supabase Storage
- **Database:** PostgreSQL
- **Notifications:** Supabase Edge Functions
- **Font:** DM Sans + DM Serif Display

## 🚀 Development

```bash
git clone https://github.com/Nomssky/carenote.git
cd carenote
npm install
npx expo start
```

## 📦 Build APK

```bash
eas build --platform android --profile production
```

## 🔄 Update OTA

```bash
eas update --branch production --message "update message"
```
